Get ollim-bot running and send your first message in 6 steps.

## Prerequisites

- **OS** — Linux, macOS, or Windows (WSL works too)
- **A terminal** — Terminal on macOS, PowerShell on Windows, `Ctrl + Alt + T` on Linux
- [Git](https://git-scm.com/) — for cloning the source code
- [uv](https://docs.astral.sh/uv/) — automatically installs the required Python version
- A [Claude](https://claude.ai) subscription (Pro or Max) — the bot authenticates via the bundled Agent SDK CLI at startup
- A Discord account

Install uv if you don't have it:

<Tabs>
  <Tab title="Linux / macOS">
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```
  </Tab>
</Tabs>

<Note>
ollim-bot is a single-user bot designed to run as a personal assistant
in Discord DMs. No server is required.
</Note>

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/Ollim-AI/ollim-bot.git
    cd ollim-bot
    uv tool install --editable .
    ```

    This installs `ollim-bot` and
    [`claude-history`](https://github.com/Ollim-AI/claude-history) as
    global commands. `claude-history` is a CLI tool that lets the bot
    search past conversations so it can remember things.

    <Tip>
    Command not found? Restart your terminal so the new commands are on your PATH. See [troubleshooting](/development/troubleshooting) for more.
    </Tip>
  </Step>

  <Step title="Create a Discord bot">
    <Tip>
    First time creating a Discord bot? See [Discord bot setup](/self-hosting/discord-bot-setup) for a step-by-step walkthrough with screenshots.
    </Tip>

    1. Go to the [Discord Developer Portal](https://discord.com/developers/applications) and click **New Application**
    2. Go to **Bot** in the sidebar, uncheck **Public Bot** (the bot is single-user — only you should install it), click **Reset Token**, and copy the token
    3. Enable **Message Content Intent** under Privileged Gateway Intents (required for the bot to read your messages)

    <Warning>The bot token is shown only once. If you lose it, you must reset it.</Warning>
  </Step>

  <Step title="Configure the bot">
    Copy the example environment file:

    <Tabs>
      <Tab title="Linux / macOS">
        ```bash
        cp .env.example .env
        ```
      </Tab>
      <Tab title="Windows (cmd)">
        ```cmd
        copy .env.example .env
        ```
      </Tab>
    </Tabs>

    Open `.env` and set the three variables:

    - `DISCORD_TOKEN` — the bot token you just copied
    - `OLLIM_USER_NAME` — your first name (the bot will call you this)
    - `OLLIM_BOT_NAME` — whatever you want to call the bot (e.g., "Ollim")
  </Step>

  <Step title="Install to your Discord account">
    1. Go to **Installation** in the sidebar — check **User Install**, uncheck **Guild Install** (the bot runs in DMs, not server channels)
    2. Set the install link to **Discord Provided Link** and add the `applications.commands` scope
    3. Copy the **Install Link** and open it in your browser — select **Add to my apps**
  </Step>

  <Step title="Start the bot">
    ```bash
    ollim-bot
    ```

    On startup, the bot checks whether Claude is authenticated. If not,
    it DMs you an OAuth link via the Discord REST API — click the link
    to sign in with your Anthropic account. The bot blocks until you
    complete sign-in. Once authenticated, the bot creates its data
    directory at `~/.ollim-bot/`, connects to Discord, and sends you a
    startup DM.

    <Tip>
    Bot not starting? See [common startup issues](/development/troubleshooting#startup-failures).
    </Tip>
  </Step>

  <Step title="Send your first message">
    Reply to the startup DM to start a conversation — responses stream
    in real time. Try one of these to see the bot in action:

    > Remind me to take out the trash tomorrow at 10am, and follow up if I haven't done it by noon

    This demonstrates smart follow-ups — the bot will check back if you
    haven't confirmed the task.

    > Set up a morning routine that gives me a summary of my day every weekday at 9am

    This sets up a proactive check-in — the feature that makes ollim-bot
    different from a regular chatbot.

    You can also try `/cost` to see token usage or `/model sonnet` to
    switch models.

    <Note>
    After [setting up Google integration](/getting-started/setup), try: "What's on my calendar today?"
    </Note>
  </Step>
</Steps>

<Tip>
Google integrations (Tasks, Calendar, Gmail) require additional OAuth
setup. See [optional integrations](/getting-started/setup) to configure
these.
</Tip>

## CLI commands

Beyond running the bot, `ollim-bot` provides CLI subcommands for
managing routines, reminders, and Google services:

```bash
ollim-bot routine list     # Show all routines
ollim-bot reminder add --delay 30 -m "take a break"
ollim-bot tasks list       # List Google Tasks
ollim-bot cal today        # Show today's calendar events
ollim-bot help             # Show all available commands
```
