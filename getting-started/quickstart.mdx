Get ollim-bot running and send your first message in under 5 minutes.

## Prerequisites

- **OS** — Linux, macOS, or Windows (WSL works too)
- **A terminal** — every command in this guide runs here. Open one with:
  - **macOS** — Spotlight (`Cmd + Space`) → type "Terminal"
  - **Windows** — Start menu → type "PowerShell"
  - **Linux** — `Ctrl + Alt + T` in most distros
- [Git](https://git-scm.com/)
- [uv](https://docs.astral.sh/uv/) (automatically installs the required Python version):

<Tabs>
  <Tab title="Linux / macOS">
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```
  </Tab>
</Tabs>

- A [Claude](https://claude.ai) subscription (Pro or Max) — the bot authenticates via the bundled Agent SDK CLI at startup
- A Discord account

<Note>
ollim-bot is a single-user bot designed to run as a personal assistant
in Discord DMs. No server is required.
</Note>

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/Ollim-AI/ollim-bot.git
    cd ollim-bot
    uv tool install --editable .
    ```

    This installs `ollim-bot` and
    [`claude-history`](https://github.com/Ollim-AI/claude-history) as
    global commands. `claude-history` is a CLI tool that lets the bot
    search past conversations so it can remember things.
  </Step>

  <Step title="Create a Discord bot">
    1. Go to the [Discord Developer Portal](https://discord.com/developers/applications) and click **New Application**
    2. Go to **Bot** in the sidebar, uncheck **Public Bot**, click **Reset Token**, and copy the token
    3. Enable **Message Content Intent** under Privileged Gateway Intents
    4. Copy the example environment file and fill in your values:

    <Tabs>
      <Tab title="Linux / macOS">
        ```bash
        cp .env.example .env
        ```
      </Tab>
      <Tab title="Windows (cmd)">
        ```cmd
        copy .env.example .env
        ```
      </Tab>
    </Tabs>

    Open `.env` and set `DISCORD_TOKEN` to your bot token, `OLLIM_USER_NAME`
    to your name, and `OLLIM_BOT_NAME` to whatever you want to call the bot.

    5. Go to **Installation** in the sidebar — check **User Install**, uncheck **Guild Install**
    6. Set the install link to **Discord Provided Link** and add the `applications.commands` scope
    7. Copy the **Install Link** and open it in your browser — select **Add to my apps**

    <Warning>The bot token is shown only once. If you lose it, you must reset it.</Warning>

    See [Discord bot setup](/self-hosting/discord-bot-setup) for the full walkthrough with screenshots.
  </Step>

  <Step title="Start the bot">
    ```bash
    ollim-bot
    ```

    On startup, the bot checks whether Claude is authenticated. If not,
    it DMs you an OAuth link via the Discord REST API — click the link
    to sign in with your Anthropic account. The bot blocks until you
    complete sign-in. Once authenticated, the bot creates its data
    directory at `~/.ollim-bot/`, connects to Discord, and sends you a
    startup DM.
  </Step>

  <Step title="Send your first message">
    Reply to the startup DM to start a conversation — responses stream
    in real time. Try `/cost` to see token usage or `/model sonnet` to
    switch models.
  </Step>
</Steps>

<Tip>
Google integrations (Tasks, Calendar, Gmail) require additional OAuth
setup. See [optional integrations](/getting-started/setup) to configure
these.
</Tip>

## CLI commands

Beyond running the bot, `ollim-bot` provides CLI subcommands for
managing routines, reminders, and Google services:

```bash
ollim-bot routine list     # Show all routines
ollim-bot reminder add --delay 30 -m "take a break"
ollim-bot tasks list       # List Google Tasks
ollim-bot cal today        # Show today's calendar events
ollim-bot help             # Show all available commands
```
